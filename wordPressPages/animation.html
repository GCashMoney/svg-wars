1: Open your SVG image in a text editor

An SVG image can be animated using a variety of languages, but for this tutorial we will be using CSS or Cascading Style Sheets. First, we will need to get the html code of the SVG image. We can do this by opening the SVG image in a text editor. Here are some free text editors available for use:
  • Notepad++
  • Sublime Text
  • Atom
Simply right-click the SVG image and open it with the text editor. For this tutorial we will be using Notepad++. Your SVG image should look something like this in the program.

2: Put the HTML and CSS in CodePen

For this tutorial, we will be using CodePen in order to test our animation. Create a profile on their home website and choose to create a new pen. Copy all of the HTML code in your SVG file and paste it in the HTML zone. Your pen should look something like this except without the CSS code:

3: Edit the HTML and CSS

Next, go through the HTML and begin changing things to find out which parts affect the paths of the image. CodePen allows the project to be a playground, so don't be afraid to go wild! You can change or add id's to the paths in order to add styles through CSS. The image below shows how the id "fillred" connects the HTML and CSS. The styles within the "fillred" id will fill animate filling the outline with color.

4: Create a new SVG file with the new HTML and CSS

While a SVG image can be animated with styles from another file, the styles can be included within the file itself so you do not have to refer to another location. Open the SVG file in a text editor, then copy the HTML from CodePen and paste all of it over the content of the SVG.
